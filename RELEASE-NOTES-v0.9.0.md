# Release notes v0.9.0 - Game-informed MCDA decision support

## Summary

v0.9.0 adds a decision-support layer to the primary care funding architecture project. The package now includes a game-informed MCDA that lets decision-makers score where each mapped game sits, weight decision criteria, compare policy options and examine sensitivity across stakeholder perspectives.

## Added

- MCDA code module and tests.
- 10 decision criteria mapped to the 14 games.
- 10 policy options.
- 6 stakeholder weight sets.
- Diagnostic game-position priority scoring.
- Policy option scorecard and risk-adjusted MCDA results.
- Sensitivity analysis across weight sets.
- Workshop guide, rubrics and templates.
- MCDA workbook, report and figures.

## Caveat

The MCDA is demonstrative and non-calibrated. It should be used to structure deliberation and validation, not as a final policy ranking.
