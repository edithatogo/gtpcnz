# Security

This repository does not currently process secrets, credentials, or private health data.

## Reporting

Please report suspected security issues privately to the repository maintainer rather than opening a public issue.

## Scope

In scope:

- Accidental secret exposure.
- Deployment configuration issues.
- Dependency vulnerabilities that affect the public site or dashboard.

Out of scope:

- Policy disagreement.
- Model interpretation disagreement without a security impact.
- Third-party source availability or content changes.
