# Release notes v1.7.0

## Full parameterised model scaffold

This release continues the calibrated-model pathway by expanding the model into a fully specified parameterisation framework.

## Added

- 70-parameter register across demand, supply, funding, governance, equity, ambulance, hospital, risk and implementation domains.
- Data-input contract for future empirical calibration.
- Scenario parameter matrix for 10 policy scenarios.
- Dynamic monthly model for 60-month simulations.
- Sensitivity analysis and calibration target matrix.
- Scenario results and plots.
- Tests for parameter bounds, scenario coverage, model ordering and sensitivity outputs.

## Interpretation

This is a fully parameterised scaffold, not a real-data calibrated predictive model.
