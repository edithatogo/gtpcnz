# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Examples of behaviour that contributes to a positive environment:

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Give and accept constructive feedback.
- Focus on what is best for the project and its users.

Examples of unacceptable behaviour:

- Harassment, intimidation, or discriminatory language.
- Personal attacks or trolling.
- Publishing private information without explicit permission.
- Sustained disruption of project work.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that are not aligned with this Code of Conduct. Serious or repeated breaches may result in temporary or permanent bans from participation.

Instances can be reported to the project maintainer. Reports will be reviewed and addressed in context.
