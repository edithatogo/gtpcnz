# Track 073 - Independent public validation source expansion

Identify and register additional independent public aggregate validation sources beyond the current Q3/Q4 PHO access workbook and capitation schedule evidence.

This track covers:

- discovery of public regional, rurality, subgroup-gradient, utilisation, workforce, hospital-pressure, and fiscal aggregate sources;
- public access, licence, checksum, retrieval, transform, schema, and validation-gate review;
- separation of independent validation targets from sources already used for calibration targets;
- explicit claim-specific boundaries before any precision or causal claim is considered.

This track does not use private administrative data, patient-level data, confidential OIA responses, stakeholder analysis, or unpublished expert elicitation.
