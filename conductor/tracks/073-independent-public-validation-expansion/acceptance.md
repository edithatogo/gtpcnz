# Acceptance

- Candidate source inventory records public access, licence, retrieval path, expected grain, and claim family.
- At least one new independent public validation source is either registered with a retrieval plan or explicitly rejected with reasons.
- Any new processed artefact has schema validation and hash sidecars.
- Strict public-only and transformed-schema checks pass.
- Documentation explains whether the source supports validation, readiness only, or rejection.
