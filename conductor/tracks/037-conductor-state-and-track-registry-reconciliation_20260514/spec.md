# Spec

## Requirements

- Reconcile `conductor/index.md`, `conductor/tracks.md`, `conductor/state.md` and track-local `metadata.json`.
- Decide whether tracks marked `active` or `planned` are still executable, complete, superseded or archived.
- Preserve the publication priorities already stated in `tracks.md`.
- Keep post-surface contract work visible without reopening completed public visual release tracks.

## Done means

Central status and track-local metadata agree, and `$conductor-status` no longer gives contradictory active-track evidence.
