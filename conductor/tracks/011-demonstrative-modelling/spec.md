# Track 011: Demonstrative modelling v0.6.0

## Purpose

Create executable, non-calibrated demonstrative models for each of the 14 mapped New Zealand policy games.

## Scope

- One model function per game, G1-G14.
- Five policy scenarios.
- Scenario-level summary outputs.
- Per-game equilibrium labels.
- One-at-a-time sensitivity around the Primary Care Benefits Schedule scenario.
- Audit matrix linking each game to code, results and next validation steps.

## Non-goals

- This is not a calibrated empirical forecast.
- This is not a cost-effectiveness analysis.
- This is not a proof of policy effectiveness.
