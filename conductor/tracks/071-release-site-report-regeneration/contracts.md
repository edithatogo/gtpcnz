# Contracts

- Generated release artefacts must be reproducible from checked-in public source snapshots and model outputs.
- Release artefacts may state `public_aggregate_validated` only for the registered aggregate validation lane.
- Release artefacts must not claim precise fiscal savings, ED reductions, hospital-demand reductions, workforce effects, implementation impacts, or causal effects.
- Manifest hashes must match the generated public source snapshot, parameter registry, model code, and output artefacts used by the site/report.
