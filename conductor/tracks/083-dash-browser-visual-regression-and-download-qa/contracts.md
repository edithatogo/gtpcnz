# Contracts

- Browser tests must not require secrets, authenticated sessions, paid services, or live external availability for local core gates.
- Visual tests must check usability and nonblank rendering, not exact decorative pixels unless snapshots are stable.
- Downloads must be validated for filename, nonempty content, and public-safe fields.
- Mobile and desktop checks must ensure text does not overlap or disappear.
- Browser QA must not change model claim wording.
