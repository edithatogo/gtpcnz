# Contracts

- Every surfaced model must include claim level, calibration status, uncertainty type, source/provenance status, and not-valid-for wording.
- Every chart must have a table fallback and downloadable CSV.
- Every live computation must obey current caps unless profiling evidence and tests justify a change.
- A model engine that is not public-safe must be listed as deferred rather than silently omitted.
- No Dash route may import Streamlit.
