# Agent Prompts

## Implementation Agent

Build the public calibration diagnostics dashboard in Dash. Use existing calibration modules and evidence ledgers, keep status language precise, and make blockers visible. Do not upgrade any claim beyond public aggregate validation/readiness status.

## Review Agent

Review the calibration diagnostics for claim creep, hidden blockers, missing provenance, missing table fallbacks, and any wording that implies linked-data, patient-level, causal, or precise forecast validation.
