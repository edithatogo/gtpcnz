# Policy brief suite

## Intended sequence

1. **New Zealand Primary Care Benefits Schedule** — the core architecture proposal.
2. **PHO functions versus PHO intermediation** — separates population-health value from payment-gateway issues.
3. **Ambulance, primary care and hospital avoidance** — brings prehospital services into the same system model.
4. **The Australian Commonwealth-state game** — comparative analysis.
5. **Top-tier KPIs for primary care and ambulance** — accountability proposal.
6. **ACC as a primary care supply stabiliser** — optional, high-value technical brief.
7. **Evidence and modelling plan** — methodological brief for RACMA and academics.

## Brief format

Each brief should be 4–6 pages and include:

- problem statement;
- why current policy may be insufficient;
- proposed reform;
- mechanism;
- evidence status;
- risks/counterarguments;
- equity implications;
- implementation pathway;
- modelling questions.
