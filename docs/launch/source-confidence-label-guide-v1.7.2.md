# Source-confidence label guide v1.7.2

Use these labels consistently in Substack posts, appendices, policy briefs and model-facing documents.

| Label | Meaning | Use in public posts |
|---|---|---|
| Public document | Directly supported by official/publicly available source | Can be stated confidently with link |
| Peer-reviewed evidence | Supported by academic literature | Can be stated with caveats about context |
| Economic theory | Standard microeconomic/game-theoretic reasoning | Frame as mechanism, not observed fact |
| Stakeholder intelligence | Email/interview/sector information not independently verified | Do not state as fact; call it a question or OIA target |
| Model assumption | Used to make a scenario executable | Do not state as real-world effect |
| Policy judgement | Normative recommendation or design preference | Own as judgement, not evidence |

## Default label for the series
Most posts use:

> public-document supported policy hypothesis, economic theory and model assumption; not calibrated prediction.

That label is deliberately conservative.
