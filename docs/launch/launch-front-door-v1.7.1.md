# Launch front-door v1.7.1

For publication and stakeholder engagement, use only these front-door artefacts first:

1. **Series landing page** — explains what the series is and is not.
2. **Post 1** — starts with the broad system question.
3. **One-page visual summary** — for busy stakeholders.
4. **Common objections and responses** — for likely pushback.
5. **Model card** — for anyone asking what the model can and cannot do.

Everything else is supporting material. Do not send the full repository to general readers unless they explicitly ask.

## Active engagement pathway
- Substack readers.
- Primary care, urgent care, ambulance, rural and equity stakeholders.
- Māori and Pacific reviewers.
- Officials and think-tank contacts where relationships already exist.
- Academic collaborators for review/modelling.

## RACMA status
RACMA is excluded from proactive outreach in this release. If RACMA comes back interested, use the decision-maker summary and model card, not the full package.
