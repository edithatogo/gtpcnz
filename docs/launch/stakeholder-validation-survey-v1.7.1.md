# Stakeholder validation survey v1.7.1

Purpose: a ten-minute survey to test whether stakeholders think the mapped games are real, harmful, tractable and risky.

## Instructions
For each game, score 1 to 5.

1 = strongly disagree / low / not important. 5 = strongly agree / high / very important.

| Question | Scale |
|---|---|
| Is this game real in New Zealand? | 1–5 |
| How harmful is the current equilibrium? | 1–5 |
| How much does it contribute to hospital demand? | 1–5 |
| How important is it for equity? | 1–5 |
| How tractable is reform? | 1–5 |
| How risky is reform? | 1–5 |
| How confident are you? | 1–5 |

## Open questions
- Which game is missing?
- Which claim is overstated?
- Which safeguard is non-negotiable?
- What would change your mind?
- What data would you trust?
