# Visual production workflow

Use this workflow for every new public diagram, plot or image.

## Steps

1. Identify the target reader question.
2. Select the visual type: conceptual diagram, microeconomics diagram, model-index plot, evidence status chart or calibration readiness chart.
3. Assign a status label.
4. Create the source file.
5. Export static formats needed for Substack, Quarto, GitHub Pages and Streamlit.
6. Write alt text.
7. Write Substack and Quarto captions.
8. Record source and output paths in the figure manifest.
9. Check the visual against the claim boundary.

## Quality gate

No public visual is complete until it has:

- source file;
- exported image;
- alt text;
- caption;
- status label;
- caveat if conceptual, toy or model-generated;
- target surface;
- source-confidence label.
