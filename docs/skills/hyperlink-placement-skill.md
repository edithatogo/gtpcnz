# Hyperlink placement skill

Use this skill when reviewing source links in public drafts.

## Rules

- Put links where the relevant claim appears.
- Link the first meaningful mention of each funding type document.
- Keep end source lists as completeness checks, not the main navigation.
- Do not overload every repeated term with the same link.
- Use source-confidence labels when the source supports only part of a claim.

## Funding type examples

- Link capitation formula sources when capitation is first discussed.
- Link scheduled fee-for-service or ACC payment sources when activity payments are first discussed.
- Link co-payment evidence when patient price/access effects are discussed.
- Link PHO/accountability sources when intermediation or place accountability is discussed.

## Done means

- No important source appears only at the end if it supports a body claim.
- Funding-type documents are linked at first relevant mention.
- High-stakes claims have nearby links.
