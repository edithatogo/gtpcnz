# Substack thumbnail skill

Use this skill when creating a post thumbnail or hero-image spec.

## Thumbnail brief

Each thumbnail needs:

- post title and slug;
- one visual metaphor;
- visual hierarchy for mobile;
- 16:9 crop and square-safe centre;
- colour palette;
- no official-looking government branding;
- no patient-identifiable imagery;
- no unsupported numerical claim;
- alt text;
- caption.

## Suggested visual language

- Funding architecture: pipes, levers, bridges, routes, ledgers.
- Capitation: base platform or fixed pool.
- Fee-for-service: metered activity with guardrails.
- Co-payment: price gate or access toll.
- Hospital spillover: pathway from blocked primary care to ED.
- Calibration: incomplete instrument panel or checklist.

## Done means

- Thumbnail prompt/spec exists.
- Image file exists or is ready to generate.
- Alt text exists.
- Caption exists.
- Post manifest references the thumbnail.
