# Figure metadata and accessibility skill

Use this skill whenever adding or reviewing public images.

## Required fields

- `figure_id`
- `title`
- `source_file`
- `export_file`
- `target_surface`
- `status_label`
- `source_confidence`
- `alt_text`
- `substack_caption`
- `quarto_caption`
- `claim_boundary`

## Alt text rules

- Describe the visible structure.
- Describe the relationship or comparison.
- Do not add claims that are not visible in the image.
- Do not say "image of" unless needed for clarity.

## Caption rules

- Captions should explain how to read the figure.
- Captions must say whether the figure is conceptual, toy, model-generated or empirical.
- Substack captions must be attached to the image block, not only placed in surrounding prose.
