# Release notes — v1.3.1

## Uncapped FFS entitlement clarification

This release clarifies the policy model: the proposal is to remove global caps on eligible primary medical activity and migrate part of primary care funding to an ACC-like fee-for-service stream, while retaining capitation and place-based accountability.
