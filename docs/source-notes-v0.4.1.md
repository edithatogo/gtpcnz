# Source notes — v0.4.1 research audit pack

## Reporting standards and audit-relevant sources

- PRISMA-ScR: use for scoping-review reporting and evidence-map transparency.
- PRISMA 2020: use if converting to a full systematic review.
- STRESS: use for empirical simulation reporting involving system dynamics or agent-based modelling.
- ODD: use for agent-based model description.
- CHEERS 2022: use if the simulation is extended into formal economic evaluation.

## Policy source anchors

The audit pack does not add new substantive policy claims beyond v0.4.0. It adds traceability and quality-control artefacts to the existing anchors:

- NZ capitation reweighting documents.
- Sapere-related briefing on capitation reweighting limitations.
- Health and Disability System Review 2020.
- Ministry PHO finance briefing 2025.
- Health NZ ambulance commissioning/performance material.
- National Primary Care Dataset and primary care access target material.
- Australian Review of General Practice Incentives.
