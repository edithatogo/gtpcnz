# Modelling overview

## Purpose

Specify an integrated modelling programme to test whether different funding architectures change primary care supply, ambulance flow and hospital growth.

## Model types

1. **Game-theoretic model:** formalises incentives and equilibrium conditions.
2. **System dynamics model:** captures feedback loops and delayed effects.
3. **Agent-based model:** captures heterogeneous patients, providers and local markets.

## Scenario families

- Status quo.
- Capitation reweighting only.
- Capitation plus bounded activity-sensitive payment.
- Demand-driven Primary Care Benefits Schedule.
- Reduced PHO intermediation.
- Broader provider activity generation.
- Ambulance treat-and-refer/hear-and-treat benefits.
- ACC fee-for-service constraint or expansion.
- Australia Commonwealth-state shared-savings model.

## Outcomes

- primary care contacts;
- wait times;
- closed books;
- provider entry/exit;
- workforce mix;
- co-payment burden;
- rural in-person capacity;
- ambulance conveyance;
- ED presentations;
- ambulatory-sensitive hospitalisations;
- total public expenditure;
- distributional equity.

## Model philosophy

The first objective is not point prediction. The first objective is mechanism testing: which assumptions would need to be true for the proposed architecture to improve access and reduce hospital pressure?
