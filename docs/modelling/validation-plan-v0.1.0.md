# Validation plan

## Model validation levels

### 1. Face validity

Review with clinicians, practice owners, nurse practitioners, pharmacists, paramedics, PHO leaders, ACC, Health NZ, Treasury and hospital managers.

### 2. Internal validity

- unit tests;
- extreme value testing;
- deterministic seed reproducibility;
- scenario consistency checks;
- sensitivity analysis.

### 3. External validity

Compare model outputs to known patterns:

- access delays;
- ED demand;
- ambulance conveyance;
- ambulatory-sensitive hospitalisations;
- provider workforce distribution;
- rural access patterns.

### 4. Policy validity

Ask whether model findings remain stable under plausible alternative assumptions and whether they help discriminate between policy options.

## Reporting

Use STRESS for simulation reporting and ODD for agent-based model reporting.
