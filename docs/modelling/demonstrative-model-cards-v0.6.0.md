# Demonstrative model cards v0.6.0

Each card records the demonstrative model status for one mapped game.

## G1: Hospital-salience budget game
- **Mechanism:** Hospital rescue bias falls only when upstream access becomes visible, funded and politically salient.
- **Highest welfare scenario:** S3 (Full upstream access architecture) with welfare 66.51.
- **Highest hospital-pressure scenario:** S0 (Status quo tight control) with pressure 81.29.
- **Highest gaming-risk scenario:** S4 (Loose benefits, weak controls) with risk 34.38.
- **Audit status:** Demonstrative only; requires calibration and stakeholder validation.
