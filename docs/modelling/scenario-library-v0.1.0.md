# Scenario library

| Scenario | Description | Main hypothesis |
|---|---|---|
| S0 | Status quo | Upstream constraint persists; hospital pressure grows |
| S1 | Capitation reweighting only | Distribution improves; total supply remains constrained |
| S2 | Capitation + bounded access payment | Marginal supply improves modestly |
| S3 | Primary Care Benefits Schedule | Marginal supply expands; cost and equity need safeguards |
| S4 | Benefits Schedule + broader provider eligibility | GP bottleneck reduces; fragmentation risk rises |
| S5 | Benefits Schedule + ambulance treat-and-refer | ED flow reduces if follow-up is reliable |
| S6 | Reduced PHO intermediation | Market entry improves if population functions preserved |
| S7 | ACC FFS constrained | Practice viability falls; non-injury supply may fall |
| S8 | Australia shared-savings | Better alignment between primary investment and hospital savings |

## Scenario outputs

For each scenario, report:

- total primary care contacts;
- contacts by provider type;
- wait time;
- co-payment distribution;
- unmet need;
- ED presentations;
- ambulance conveyance;
- hospital admissions;
- provider entry/exit;
- fiscal cost;
- equity distribution.
