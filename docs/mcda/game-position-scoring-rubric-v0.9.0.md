# Game-position scoring rubric v0.9.0

This rubric is used before ranking policy options. It asks decision-makers to locate each mapped game in the current system.

## Questions for each game

| Field | Scale | Question |
|---|---:|---|
| Harm if unresolved | 1-5 | How damaging is the current equilibrium if left unresolved? |
| Hospital growth driver | 1-5 | How strongly does this game channel unmet need or resources toward hospitals? |
| Equity relevance | 1-5 | How strongly does the game affect equity, Te Tiriti legitimacy, trust or high-need groups? |
| Reform tractability | 1-5 | How feasible is it to shift the game through policy? |
| Reform risk | 1-5 | How risky is reform if poorly designed? |
| Confidence | 1-5 | How confident are we in our assessment? |

## Workshop use

The facilitator should invite disagreement. A low score is not a failure. It indicates either lower priority, lower confidence, or a need for better evidence. The most useful output is often the disagreement pattern between stakeholder groups.
