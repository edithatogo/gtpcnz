# Contract-compliance audit — v1.2.1

## Overall result

The package now meets the accumulated contract requirements for the current phase: a policy-research and publication-ready package that lays out the background and theory, maps the broader New Zealand policy games, models them demonstratively, synthesises a hybrid model, provides MCDA decision support, produces reports and policy briefs, and includes a human-readable Substack publication layer.

The package remains intentionally **not** a fully empirically calibrated predictive model. That is a future phase, not a requirement for current release.

## Substack publication check

The Substack-ready layer now includes:

- 12 post drafts;
- one or more figures per post;
- at least five unique external source links per post;
- average sentence length at or below 16 words;
- a source bank;
- a plain-English glossary;
- a voice/style guide;
- figure placement and alt-text guidance.

The posts are best described as **Substack-ready drafts for Dylan’s final editorial pass**. They are readable, humanised and source-linked, but Dylan should still do the final judgement pass on voice, political sharpness, Te Tiriti/Māori and Pacific nuance, and stakeholder risk.

## Modelling check

The package includes:

- mapped 14-game New Zealand policy atlas;
- demonstrative model functions for each mapped game;
- uncertainty/sensitivity layer;
- final hybrid synthesis model;
- MCDA decision-support layer;
- source-informed parameterised model run;
- validation pathway that deliberately avoids overclaiming predictive calibration.

## Main caveats

- This package should not be used to claim precise quantitative effects on emergency department demand, admissions, costs or workforce supply.
- External links were not exhaustively live-HTTP checked; link formatting, source count and local Markdown references were checked.
- Stakeholder validation and rapid scoping review execution remain next-stage work.
