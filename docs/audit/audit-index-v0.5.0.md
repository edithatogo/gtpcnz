## Audit summary

The v0.5.0 package is suitable for:

- RACMA colleague review;
- research collaborator recruitment;
- ethics/governance scoping;
- review protocol finalisation;
- OIA/data request management;
- first synthetic simulation build.

It is not suitable for claiming empirical effect sizes or policy certainty.
