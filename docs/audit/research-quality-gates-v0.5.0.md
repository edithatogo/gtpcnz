# Research quality gates v0.5.0

This file translates the audit into explicit pass/partial/fail gates. It is intended for collaborator review and project governance.

## Gate interpretation

- **Pass** means the artefact exists and is internally coherent.
- **Pass with caveats** means the artefact is sufficient for scoping, but a known dependency remains.
- **Partial** means the artefact is usable but not decision-grade.
- **Not yet passed** means the work should not be represented as empirically validated.

The current package passes conceptual and protocol gates, but not empirical, equity, fiscal or stakeholder-validation gates.

See `docs/audit/research-quality-gates-v0.5.0.csv` for the machine-readable table.
