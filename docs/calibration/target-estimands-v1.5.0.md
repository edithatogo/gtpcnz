# Target estimands for the future predictive model v1.5.0

The predictive model should estimate the effect of moving from the current reform pathway to a hybrid architecture with capitation plus uncapped scheduled fee-for-service for eligible primary medical activity plus place-based accountability.

Primary estimands:

- change in seven-day primary care access;
- change in primary medical contacts per 1,000 population;
- change in unmet need;
- change in emergency department presentations;
- change in ambulance conveyance and non-conveyance with safe follow-up;
- change in potentially avoidable admissions;
- change in public cost and patient co-payment burden;
- distributional effects by ethnicity, deprivation, age, rurality and multimorbidity.
