# Draft OIA request: current capitation formula and calculation basis

To: Ministry of Health / Health New Zealand

Dear Sir/Madam,

I request under the Official Information Act 1982 the following information relating to current and proposed general practice capitation funding:

1. The current full capitation calculation formula used for general practice funding.
2. All current rate tables, weights, age-sex bands, deprivation adjustments, rural adjustments, high-needs adjustments and other variables used in calculating capitation payments.
3. Any calculation workbooks, code, spreadsheets or technical documentation used to operationalise the current formula.
4. All current capitation-related funding streams, including but not limited to FLS, VLCA, CSC-related funding, Care Plus, SIA, rural funding and any top-ups or add-on payments.
5. Any PHO Services Agreement schedules or appendices that specify capitation payment calculation, payment flow, pass-through requirements or provider eligibility.
6. The technical documentation for the proposed capitation reweighting from 1 July 2026, including modelling assumptions, coefficients, weights, variables, implementation options and transition arrangements.
7. Any advice since 1 January 2024 comparing capitation reweighting with more fundamental changes to the primary care funding model.

If any material is withheld, please identify the specific document or component withheld and the grounds for withholding.

Yours sincerely,

Dylan A Mordaunt
