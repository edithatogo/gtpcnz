# Public Wording Contract v1.8.4

Canonical phrasing contract for all GTPCNZ public surfaces.

## 1. Public Caveat

> This is a public-data anchored benchmark and educational explainer. It is not linked-data calibrated and not a patient-level forecast. It should not be used to claim precise fiscal savings, hospital-demand reductions, workforce effects, or implementation impacts.

## 2. Status Labels

| Label | Meaning |
|---|---|
| Educational explainer | Simplified teaching controls, not the model forecast. |
| Model-generated index | From benchmark logic, not observed NZ outcomes. |
| Evidence readiness | Tracks what data exists and what is still needed. |
| Calibration readiness | Describes steps needed before empirical calibration. |

## 3. Uncapped Does Not Mean Uncontrolled

Uncapped does not mean uncontrolled. It means scheduled, rules-based, audited, clinically governed and place-accountable.
