# Source-read catalogue v1.4.0

This catalogue separates documents that were publicly sourced and read from documents that are withheld, internal, anecdotal, or not yet verified. “Read” here means read through the available public page, public PDF search text, or official release excerpt available to the web tool. It does not mean all redacted or withheld content has been accessed.

## SRC001 — Capitation reweighting public page

- **Status:** Sourced and read from public page
- **Type:** Public policy page

- **URL:** https://www.health.govt.nz/strategies-initiatives/programmes-and-initiatives/primary-and-community-health-care/capitation-reweighting

- **Key points read:** Capitation was introduced in 2002; is core funding for general practice; current formula based on late-1990s use; new formula considers age, sex, multimorbidity, rurality and deprivation; expected from 1 July 2026 subject to PHO Services Agreement negotiation.
- **Implication for analysis:** Supports background; also shows current reform is allocation-oriented and still capitation-centred.

## SRC002 — Cabinet paper: Primary Health Care Funding Improvements

- **Status:** Sourced and read from public release/search-rendered PDF text
- **Type:** Cabinet paper

- **URL:** https://www.health.govt.nz/system/files/2025-07/Cabinet%20paper%20-%20Primary%20Health%20Care%20Funding%20Improvements%20Redacted.pdf

- **Key points read:** States NZ already has a blended model of capitation, co-payments and targeted streams; capitation totalled $915.7m in 2024/25; structure is complicated; poor visibility of what funding buys; reweighting based on >2m patients across 18 PHOs.
- **Implication for analysis:** Requires package to acknowledge the Government’s own description of a blended model and poor funding visibility.

## SRC004 — Briefing H2025068320: Capitation reweighting – further advice
