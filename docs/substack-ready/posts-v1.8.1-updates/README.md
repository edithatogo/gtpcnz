# Substack v1.8.1 update pack

This folder contains additive update notes for the existing public Substack series. These files are intended to update, annotate, or append to the published v1.6.0 series. They do not replace the original post files.

Use these notes as versioned update boxes or follow-up sections when refreshing the Substack publication surface. The baseline posts remain in `docs/substack-ready/posts-v1.6.0-public/`.

## Claim boundary

The v1.8.1 update changes the public evidence status from a purely demonstrative model to a public aggregate validation lane for registered gates only. It does not make the model a linked-data calibrated model, patient-level forecast, implementation impact estimate, or causal impact evaluation.

Do not convert these update notes into claims of precise fiscal savings, ED reductions, hospital-demand reductions, workforce effects, implementation impacts, or causal effects.
