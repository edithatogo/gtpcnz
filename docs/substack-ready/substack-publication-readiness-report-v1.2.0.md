# Substack publication readiness report - v1.2.0

## Status

The previous Substack materials were scaffold-length drafts. This release expands them into publication-ready drafts for editorial review.

The series now includes:

- 12 expanded posts;
- 7 new conceptual diagrams;
- 5 reused model/output plots suitable for visual explainers;
- a source bank;
- a plain-English glossary;
- a voice and style guide;
- figure placement and alt text;
- a post-by-post quality assurance checklist.

## Editorial caveat

This pack is ready to use as a publication draft set, but it should not be posted without a final human pass by Dylan. The key final checks are political judgement, phrasing around Primary Health Organisations, co-payment safeguards, Māori and Pacific provider framing, and how direct to be about Health New Zealand and Treasury incentives.
