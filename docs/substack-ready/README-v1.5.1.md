# Substack publication pack v1.5.1

This pack contains short public posts plus optional deep-dive appendices.

- `posts-v1.5.1-public/`: shorter public posts, generally 750-1,050 words.
- `appendices-v1.5.1/`: optional deep-dive appendices.
- `long-drafts-v1.5.1-backup/`: original v1.5.0 long drafts retained as backup.
- `complete-substack-series-v1.5.1.md`: all short posts in one file.
- `complete-substack-appendices-v1.5.1.md`: all appendices in one file.
- `short-form-publication-strategy-v1.5.1.md`: posting strategy and cadence.
- `substack-post-qa-checklist-v1.5.1.csv`: word-count and link-count QA.

Use the short post as the public article, then link to the appendix only where readers want the detail.
