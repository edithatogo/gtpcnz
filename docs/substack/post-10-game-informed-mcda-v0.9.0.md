# Substack post 10 - How should we decide which primary care funding game matters most?

The next step in this series is not another argument about whether capitation or fee-for-service is better.

It is a decision question.

Which strategic traps matter most? Which are we most confident about? Which are easiest to shift? Which are dangerous if we shift them badly?

That is where a game-informed MCDA is useful. MCDA means multi-criteria decision analysis. In plain English, it is a structured way of comparing options when there is no single metric that can carry the whole decision.

In primary care funding, there is no single metric. Access matters. Equity matters. Rural presence matters. Hospital avoidance matters. Fiscal control matters. Gaming risk matters. Clinical governance matters. Political feasibility matters. Data matters.

The game map tells us what the traps are. The MCDA asks which traps matter most.

The most important point is this: the preferred model is not simply demand-driven funding. A loose benefits schedule can increase access while creating gaming, safety, fiscal and equity risks.

The better line is:

> Demand-driven within rules; not demand-driven without rules.

That means defined contact types, provider scope, clinical governance, co-payment protections, data visibility, audit, top-tier KPIs, and retained equity and locality functions.

The MCDA does not prove the answer. It shows the trade-offs. That is exactly what a serious policy debate should do.
