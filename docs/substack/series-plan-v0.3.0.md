# The Primary Care Funding Game

## Series purpose

To build public understanding before arguing for a specific policy reform. The series should explain economics, incentives and system design in plain language, while remaining transparent about uncertainty.

## Central line

> Stop tightly rationing primary care and ambulance activity, then wondering why hospitals grow.

Use sparingly. The professional version is:

> Current reform improves allocation within capitation. The larger question is whether the architecture constrains lower-cost access and shifts avoidable demand into higher-cost hospital care.
