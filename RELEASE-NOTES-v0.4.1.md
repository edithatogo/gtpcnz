# Release notes — v0.4.1

This release adds a research audit and traceability pack for the New Zealand game-theory mapping work.

## Purpose

The mapping work had reached a level of complexity where external reviewers need a clear audit trail. This release documents:

- which artefacts exist;
- which claims are being made;
- which claims are supported by public evidence, theory, modelling assumptions or policy hypothesis;
- which assumptions remain to be tested;
- what would falsify or weaken the hypothesis;
- how the work should be reported if it becomes a review, simulation study or economic evaluation.
